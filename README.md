## Tech stack
* Node
* Vanilla JavaScript

## Getting started

```sh
git clone https://github.com/PiotrBerebecki/custom-video-player.git
cd custom-video-player
npm install

#1. Start the app
npm start

#2. Open this url in your browser: http://localhost:3000
```
